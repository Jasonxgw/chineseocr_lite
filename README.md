## 本项目基于[chineseocr_lite](https://github.com/ouyanghuiyu/chineseocr_lite)进行接口封装

# 环境
- pytorch
- python3
  
# 运行
``` bash
python app.py
```
访问
http://localhost:8888/

# 说明
1. 调用`ocr.result()`可进行文本预测
2. 若`pse.pyd`报错则需要参考配置文章
